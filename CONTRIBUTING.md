# Contributing to Awesome Italian Music Resources

Thank you for your interest in contributing! 🎵

## How to contribute

1. Fork this repository
2. Add your resource in the correct category in README.md
3. Make sure the resource is:
   - Free or has a meaningful free tier
   - Working (check the link)
   - Relevant to music creation or Italian music culture
4. Submit a Pull Request with a short description of what you added

## Guidelines

- No spam or self-promotion without genuine value
- Prefer open-source or free resources
- Links must be direct and functional

## Questions?

Open an issue and we'll get back to you.
